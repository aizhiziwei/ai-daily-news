## 📡 这是什么

你是否也有这样的困扰？

> 每天打开 Hacker News、TechCrunch、各大 AI 实验室博客...翻了一圈英文长文，时间花了不少，真正有价值的信息却没几条。

**AI Daily News** 帮你解决这个问题。

我们每天自动从 **Andrej Karpathy 推荐的 90+ 全球顶级技术博客** 中抓取最新文章，经过 AI 多维度评分筛选，只保留真正有价值的内容，翻译成中文并生成一份 **5 分钟就能读完的精选日报**。

**覆盖信源**：OpenAI、Google DeepMind、Anthropic、Meta AI、Hacker News、TechCrunch、独立研究者博客等。
