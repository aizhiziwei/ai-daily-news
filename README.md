## 📡 这是什么

每天自动从 **Andrej Karpathy 推荐的 90+ 顶级技术博客** 抓取最新文章，通过 AI 筛选、翻译、摘要，生成一份中文日报。

**信息源覆盖**：OpenAI、Google DeepMind、Anthropic、Meta AI、Hacker News、TechCrunch 等。
